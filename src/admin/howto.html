BOND URL

F.eks. nodekø med id 5842 kan redigeres i BOND Controlroom på følgende URL:

http://edit.berlingskemedia.net/admin/content/nodequeue/5842/edit

kan anvendes i nyhedsbrevene ved at omskrive URL til

http://edit.berlingskemedia.net/bondapi/nodequeue/5842.ave-json?image_preset=620x355-c

Detaljer:

  • id skal ændres alt efter hvilken nidekø der skal hentes fra.
  • Miljønavn edit.berlingskemedia.net kan ændres alt efter hvorfra artiklerne skal hentes.
  • image_preset=620x355-c angiver hvilke billedest&oslah;rrelser der skal anvendes. Angives dette ikke, vil BOND give billeder i fuld størrelse. Se en liste over billede størrelser her: Image formats on BOND
  • Resten af URL'en skal bibeholdes som angivet ovenfor.
